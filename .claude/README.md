## Tech Stack

- **Framework:** React Native + Expo
- **Language:** TypeScript
- **Styling:** NativeWind (TailwindCSS)
- **Backend:** Firebase (Auth, Firestore, Storage)
- **AI:** OpenAI GPT-3.5
- **Maps:** Google Maps API
- **Media:** Cloudinary

## AI Assistant Features

The AI Health Guide provides:
- Symptom pattern analysis
- Disease predictions with confidence percentages
- Severity level indicators
- Symptom frequency tracking
- Health guidance (not medical diagnosis)

## Development Guidelines

1. Always maintain TypeScript type safety
2. Follow React Native best practices
3. Use NativeWind for styling consistency
4. Implement proper error handling
5. Add loading states for async operations
6. Include accessibility features
7. Test on both iOS and Android platforms
